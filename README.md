# Projeto Todo List

Uma aplicação de lista de tarefas (To-Do List) construída do zero com JavaScript puro, como parte do aprendizado de manipulação do DOM, eventos, estado e persistência no navegador.

## Funcionalidades

- **Adicionar tarefas** — Criar novas tarefas com validação de entrada
- **Remover tarefas** — Excluir tarefas individuais
- **Concluir tarefas** — Marcar/desmarcar tarefas como concluídas com contador atualizado
- **Editar tarefas** — Alterar o texto de tarefas existentes com validação
- **Filtros** — Visualizar todas, pendentes ou concluídas
- **Persistência** — Dados salvos no `localStorage` (em desenvolvimento)
- **Drag and Drop** — Reordenar tarefas (desafio futuro)

## Como rodar

Abra o arquivo `index.html` em qualquer navegador moderno. Nenhuma dependência ou build necessária.
